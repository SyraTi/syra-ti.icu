利用Overflow:Hidden实现自适应左右两列布局/填充父级元素剩余宽度

发表于 2017-07-27 更新于 2022-01-27 阅读次数： 本文字数： 35 阅读时长 ≈ 1 分钟

HTML

<div class="parent">
    <div class="Left"></div>
    <div class="Right"></div>
</div>

左侧CSS

.Left{
    width:auto;/* 左侧宽度任意 */
    float:left;
}

右侧CSS

.Right{
    overflow:hidden;
}
